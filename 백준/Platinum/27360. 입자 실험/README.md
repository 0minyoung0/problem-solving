	<li>좌측 그림: 2행 1-2열과 3행 1-2열에 스위치를 각각 설치한다.</li>
	<li>중앙 그림: 2행 2-3열과 3행 2-3열에 스위치를 각각 설치한다.</li>
	<li>우측 그림: 스위치를 추가로 설치하지 않는다.</li>
	<li>이 예제에서 편의상 두 개의 스위치를 AA와 BB로 표현했지만 스위치는 모두 동일하게 생겼기 때문에 구분이 가지 않는다.</li>
</ul>
